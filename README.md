## 为什么做这个

用户在小红书搜"平价眼霜"，看了十几篇笔记，
心里已经决定买哪个了——然后跳去淘宝搜。

小红书做了最难的事（帮用户做决策），
但没拿到钱（用户在别的地方买）。

我想解决这个问题。

---

## 具体是什么功能

在搜索结果页里，笔记和笔记之间，
插一张卡片：AI 扫描了这个搜索词下的所有笔记，
告诉你"大家提得最多的是这三个产品，价格是这些，
点这里直接买"。

用户不用跳出去，在小红书里就能完成从"看攻略"到"下单"。

---

## 和小红书现有功能有什么不同

小红书有商城，但那是一个独立入口，用户要主动去找。
小红书有"问一问"，但现在只作为内容整合工具。

这张卡片是插在搜索场景里的，
这是用户消费意图最强的那一刻，
而这个功能不需要用户主动跳转，直接在小红书平台上进行交易。

---

## 怎么用这个 Demo

1. 打开链接
2. 搜索任何你感兴趣的词（护肤品、旅游目的地、美食都行）
3. 看搜索结果里插入的那张卡片

数据是 AI 模拟的，主要用于展现产品的逻辑和进行产品演示
